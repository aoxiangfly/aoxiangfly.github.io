简单几步，实现WordPress自动提交文章到百度功能

发表于 2018-08-06

简介

  自动推送文章链接到百度对于百度优化是很重要的功能，能够加快文章的收录，有利于网站的排名的提升。网上有很多WordPress相关的插件可以实现该功能，但使用插件有一个弊端，会影响网站的加载速度。如果利用WordPress内置的函数来实现，不仅可以降低安装插件带来的影响，也能够更灵活的实现功能。

函数

  add_action是WordPress内置的函数，它有一系列动作接口，可以实现当WordPress执行某种动作时就调用指定的函数。百度自动提交功能通过调用publish_post动作接口，当发布文章的时候就会执行指定的函数将文章链接提交到百度接口。

/*
add_action( string $tag, callable $function_to_add, int $priority = 10, int $accepted_args = 1 );
具体的函数使用请看这里:https://developer.wordpress.org/reference/functions/add_action/
*/
add_action( 'publish_post', 'post_published_notification', 10, 2 );//添加在发布文章时的活动

代码

  将如下代码复制到WordPress主题内的functions.php文件内即可实现对应的功能。

//该函数可以实现自动提交文章到百度站长工具
if(!function_exists('Baidu_Submit')){
    function Baidu_Submit($post_ID) {
        $WEB_TOKEN  = 'xxxxxxxxxx';  //这里请换成你的网站的百度主动推送的token值
        $WEB_DOMAIN = get_option('home');

        //已成功推送的文章不再推送
        if(get_post_meta($post_ID,'Baidusubmit',true) == 1) return;
        $url = get_permalink($post_ID);
        $api = 'http://data.zz.baidu.com/urls?site='.$WEB_DOMAIN.'&token='.$WEB_TOKEN;
        $request = new WP_Http;
        $result = $request->request( $api , array( 'method' => 'POST', \
        'body' => $url , 'headers' => 'Content-Type: text/plain') );
        $result = json_decode($result['body'],true);

        //如果推送成功则在文章新增自定义栏目Baidusubmit，值为1
        if (array_key_exists('success',$result)) {
            add_post_meta($post_ID, 'Baidusubmit', 1, true);
        }
    }
    add_action('publish_post', 'Baidu_Submit', 0);
}

//该函数可以实现自动提交文章到百度熊掌号
if(!function_exists('Baidu_Xiongzhang_Submit')){
    function Baidu_Xiongzhang_Submit($post_ID) {

        //这里请换成你的百度推送API地址
        $api = 'http://data.zz.baidu.com/urls?appid=your-appid&token=your-token&type=realtime';

        //已成功推送的文章不再推送
        if(get_post_meta($post_ID,'BaiduXiongzhangsubmit',true) == 1) return;
        $url = get_permalink($post_ID);
        $request = new WP_Http;
        $result = $request->request( $api , array( 'method' => 'POST', \
        'body' => $url , 'headers' => 'Content-Type: text/plain') );
        $result = json_decode($result['body'],true);

        //如果推送成功则在文章新增自定义栏目BaiduXionggzhangsubmit，值为1
        if (array_key_exists('success_realtime',$result)) {
            add_post_meta($post_ID, 'BaiduXiongzhangsubmit', 1, true);
        }
    }
    add_action('publish_post', 'Baidu_Xiongzhang_Submit', 0);
}

总结

  WordPress内置API功能非常强大，能够自定义实现各种需求。当你对于自己的网站不满意，想要定制化自己需求的时候，不妨阅读一下WordPress官方文档，查查相关的API，以简单的函数替代插件更灵活的实现需要的功能。
